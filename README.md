快速开始
复制
# 克隆仓库
git clone https://github.com/imai-work/imai-work.git

# 进入项目目录
cd imai-work

# 复制并修改环境变量配置
cp .env.example .env

# 使用 Docker 启动应用
docker compose up -d
启动完成后，访问 http://localhost:3000 进入系统。
